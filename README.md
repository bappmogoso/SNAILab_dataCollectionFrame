# SNAILab_dataCollectionFrame

The data collection frame primarily mounts the sensor system comprising a ZED2i camera, TI IWR1443, and TI DCA1000 to its front. All of the parts were used and or redesigned from the <a href="https://radarml.github.io/red-rover/">RadarML Red Rover</a>.

Before commiting to any hardware work, first the following software applications must be downloaded:
  1. <a href="https://www.stereolabs.com/developers/release">ZED SDK 5.3</a>
  2. <a href="https://www.ti.com/tool/UNIFLASH">Uniflash</a>
  3. <a href="https://www.ti.com/tool/MMWAVE-STUDIO">mmWave Studio</a>
  4. <a href="https://www.ti.com/tool/CCSTUDIO">Code Composer Studio</a>
  5. <a href="https://www.ti.com/tool/MMWAVE-SDK">mmWave SDK</a>
Download and 3D print all .STL parts.
